# Changelog

All notable changes to the Mini DB project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0-milestone-6] - 2026-04-10
### Added
- `Catalog` class for managing table schemas.
- `Executor` class for interpreting AST and performing storage operations.
- Support for `CREATE TABLE`, `INSERT`, and `SELECT *`.
- End-to-end integration tests in `tests/integration_test.cpp`.
